# vista-accessory-gen
# 🏷️ Accessory Price Generator (Pro Edition)

**Professional price list generator for flooring accessories. Fast, clean, and optimized for A5 landscape printing.**

---

### 🌍 English Description
Automate the creation of technical price lists for flooring components (corners, end caps, connectors). This tool generates professional A5 layouts with automated formatting.

* **Multi-Brand Support:** Dedicated profiles for Tis, LM 55, and Indo.
* **Smart PDF Export:** Generates high-quality PDF in landscape orientation (A5 format on A4 sheet).
* **Dynamic Content:** Editable dates and automatic price formatting in UAH.
* **Modern UI:** Built with Tailwind CSS for high readability on tablets and PCs.

---

### 🇺🇦 Опис (Українська)
Професійний інструмент для швидкої генерації прайс-листів на комплектуючі для підлоги. Створено для полегшення роботи у відділах покриття.

#### 🚀 Можливості:
* **Підтримка брендів:** Окремі варіанти для Тіс (Україна) та Arbiton LM/Indo (Польща).
* **Експорт у PDF:** Автоматичне формування альбомного макета А5, готового до друку.
* **Гнучкість:** Можливість редагувати дату актуальності прямо у прев'ю.
* **Якість:** Чітке форматування цін та автоматичне округлення.

#### 🛠 Як користуватися:
1. Оберіть потрібний бренд (Тіс, LM або Indo).
2. Введіть актуальні ціни на комплектуючі.
3. Натисніть на дату внизу, щоб змінити її (якщо потрібно).
4. Використовуйте кнопки **«Відкрити PDF»** або **«Завантажити PDF»** для друку.
